# Production Checklist — Делифон MVP

## Инфраструктура

- [ ] PostgreSQL 15+ на российском хостинге (242-ФЗ)
- [ ] Redis для сессий и Celery broker
- [ ] S3-совместимое хранилище (MinIO / Selectel) для фото
- [ ] Reverse proxy (nginx / Caddy) с TLS
- [ ] Docker Swarm / K8s для оркестрации
- [ ] Автоматические бэкапы БД (ежедневно, ротация 30 дней)

## Безопасность

- [ ] Все секреты в переменных окружения (не в коде)
- [ ] AES-256 шифрование паспортных данных в БД
- [ ] HMAC-верификация ЮKassa webhook'ов
- [ ] Rate limiting на auth-эндпоинтах (5 попыток / мин)
- [ ] CORS ограничен доменами приложений
- [ ] CSP-заголовки на фронтендах
- [ ] Аудит-лог для всех админских действий

## Мониторинг

- [ ] Health-check эндпоинт `/healthz` для каждого сервиса
- [ ] Метрики (Prometheus / Grafana): CPU, RAM, latency, error rate
- [ ] Алерты: 5xx > 1%, latency p95 > 2s, БД connections > 80%
- [ ] Centralized logging (Loki / ELK)
- [ ] Sentry для трекинга ошибок

## Платежи (ЮKassa)

- [ ] Боевые ключи ЮKassa (не тестовые)
- [ ] Webhook URL зарегистрирован в ЮKassa
- [ ] Идемпотентность всех платёжных операций проверена
- [ ] Категория магазина: «аренда/прокат»
- [ ] Холды настроены (до 180 дней)

## Уведомления

- [ ] SMS-провайдер подключён (регистрация отправителя)
- [ ] Push-уведомления (FCM) настроены
- [ ] Email-рассылка (актыа, чеки) через транзакционный сервис

## Перед запуском

- [ ] Нагрузочное тестирование (100 concurrent users)
- [ ] Пенетрационное тестирование (OWASP Top 10)
- [ ] Согласие на обработку ПД в клиентском приложении
- [ ] Политика конфиденциальности и оферта опубликованы
- [ ] Seed-данные удалены из production БД
- [ ] Alembic миграции применены (`make migrate`)
