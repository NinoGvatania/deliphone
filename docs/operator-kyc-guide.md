# Руководство оператора: верификация KYC

## Общие правила

1. Проверяй паспорт в течение 15 минут после подачи заявки.
2. Сверяй фото документа с селфи клиента — лицо должно совпадать.
3. Убедись, что срок паспорта не истёк и нет следов подделки.
4. При сомнениях — отклоняй с указанием причины; клиент сможет повторить.

## Чек-лист проверки

- [ ] ФИО читаемы, совпадают с введёнными данными
- [ ] Фото паспорта чёткое, все углы видны
- [ ] Серия и номер не заретушированы
- [ ] Дата рождения — клиенту 18+
- [ ] Селфи: лицо совпадает с фото в паспорте
- [ ] Нет признаков фотошопа (тени, шрифты, текстура бумаги)

## Отказ

При отклонении обязательно выбери причину:
- `blurry` — нечёткое фото
- `mismatch` — данные не совпадают
- `expired` — просроченный документ
- `underage` — клиенту менее 18 лет
- `fraud_suspicion` — подозрение на подделку

## Безопасность (152-ФЗ)

- Не сохраняй скриншоты паспортов на личные устройства.
- Доступ к персональным данным логируется в audit_log.
- Фото паспортов хранятся в зашифрованном S3-бакете; просмотр — только через админку.
